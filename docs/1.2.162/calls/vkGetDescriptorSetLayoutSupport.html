 Categories vkGetDescriptorSetLayoutSupport Query whether a descriptor set layout can be created Syntax

Parameters device VkDevice is the logical device that would create the descriptor set layout. pCreateInfo VkDescriptorSetLayoutCreateInfo is a reference to a VkDescriptorSetLayoutCreateInfo structure specifying the state of the descriptor set layout object. pSupport VkDescriptorSetLayoutSupport is a reference to a VkDescriptorSetLayoutSupport structure, in which information about support for the descriptor set layout object is returned.
