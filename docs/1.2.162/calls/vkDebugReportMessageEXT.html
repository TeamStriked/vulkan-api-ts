 Categories vkDebugReportMessageEXT Inject a message into a debug stream Syntax 
void vkDebugReportMessageEXT();
Parameters instance VkInstance is the debug stream's VkInstance. flags VkDebugReportFlagBitsEXT specifies the VkDebugReportFlagBitsEXT classification of this event/message. objectType VkDebugReportObjectTypeEXT is a VkDebugReportObjectTypeEXT specifying the type of object being used or created at the time the event was triggered. object BigInt Number is the object where the issue was detected. object can be VK_NULL_HANDLE if there is no object associated with the event. location BigInt Number is an application defined value. messageCode Number is an application defined value. pLayerPrefix String is the abbreviation of the component making this event/message. pMessage String is a string detailing the trigger conditions.
