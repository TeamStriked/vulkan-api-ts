 Categories vkSetPrivateDataEXT Associate data with a Vulkan object Syntax 
VkResult vkSetPrivateDataEXT();
Parameters device VkDevice is the device that created the object. objectType VkObjectType is a VkObjectType specifying the type of object to associate data with. objectHandle BigInt Number is a handle to the object to associate data with. privateDataSlot VkPrivateDataSlotEXT is a handle to a VkPrivateDataSlotEXT specifying location of private data storage. data BigInt Number is user defined data to associate the object with. This data will be stored at privateDataSlot.
