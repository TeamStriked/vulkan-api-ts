 Categories vkDestroyDebugUtilsMessengerEXT Destroy a debug messenger object Syntax 
void vkDestroyDebugUtilsMessengerEXT();
Parameters instance VkInstance is the instance where the callback was created. messenger VkDebugUtilsMessengerEXT is the VkDebugUtilsMessengerEXT object to destroy. messenger is an externally synchronized object and must not be used on more than one thread at a time. This means that vkDestroyDebugUtilsMessengerEXT must not be called when a callback is active. pAllocator VkAllocationCallbacks controls host memory allocation as described in the <> chapter.
