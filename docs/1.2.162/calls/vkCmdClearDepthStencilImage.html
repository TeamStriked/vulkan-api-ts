 Categories vkCmdClearDepthStencilImage Fill regions of a combined depth/stencil image Syntax 
void vkCmdClearDepthStencilImage();
Parameters commandBuffer VkCommandBuffer is the command buffer into which the command will be recorded. image VkImage is the image to be cleared. imageLayout VkImageLayout specifies the current layout of the image subresource ranges to be cleared, and must be VK_IMAGE_LAYOUT_GENERAL or VK_IMAGE_LAYOUT_TRANSFER_DST_OPTIMAL. pDepthStencil VkClearDepthStencilValue is a reference to a VkClearDepthStencilValue structure containing the values that the depth and stencil image subresource ranges will be cleared to (see <> below). rangeCount Number is the number of image subresource range structures in pRanges. pRanges Array[VkImageSubresourceRange] is an array of VkImageSubresourceRange structures describing a range of mipmap levels, array layers, and aspects to be cleared, as described in <>.
