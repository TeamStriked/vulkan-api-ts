Parameters object When instantiated with an object, members can be filled directly. Properties VkBindSparseInfo.prototype.sType VkStructureType is the type of this structure. VkBindSparseInfo.prototype.pNext null is null or a reference to a structure extending this structure. VkBindSparseInfo.prototype.waitSemaphoreCount Number is the number of semaphores upon which to wait before executing the sparse binding operations for the batch. VkBindSparseInfo.prototype.pWaitSemaphores Array[VkSemaphore] is an array of semaphores upon which to wait on before the sparse binding operations for this batch begin execution. If semaphores to wait on are provided, they define a <>. VkBindSparseInfo.prototype.bufferBindCount Number is the number of sparse buffer bindings to perform in the batch. VkBindSparseInfo.prototype.pBufferBinds Array[VkSparseBufferMemoryBindInfo] is an array of VkSparseBufferMemoryBindInfo structures. VkBindSparseInfo.prototype.imageOpaqueBindCount Number is the number of opaque sparse image bindings to perform. VkBindSparseInfo.prototype.pImageOpaqueBinds Array[VkSparseImageOpaqueMemoryBindInfo] is an array of VkSparseImageOpaqueMemoryBindInfo structures, indicating opaque sparse image bindings to perform. VkBindSparseInfo.prototype.imageBindCount Number is the number of sparse image bindings to perform. VkBindSparseInfo.prototype.pImageBinds Array[VkSparseImageMemoryBindInfo] is an array of VkSparseImageMemoryBindInfo structures, indicating sparse image bindings to perform. VkBindSparseInfo.prototype.signalSemaphoreCount Number is the number of semaphores to be signaled once the sparse binding operations specified by the structure have completed execution. VkBindSparseInfo.prototype.pSignalSemaphores Array[VkSemaphore] is an array of semaphores which will be signaled when the sparse binding operations for this batch have completed execution. If semaphores to be signaled are provided, they define a <>. Default Properties VkBindSparseInfo.prototype.memoryBuffer ArrayBuffer Native memory reference of the structure.
