Properties VkPhysicalDeviceAccelerationStructurePropertiesKHR.prototype.sType VkStructureType is the type of this structure. VkPhysicalDeviceAccelerationStructurePropertiesKHR.prototype.pNext null is null or a reference to a structure extending this structure. VkPhysicalDeviceAccelerationStructurePropertiesKHR.prototype.maxGeometryCount BigInt Number is the maximum number of geometries in the bottom level acceleration structure. VkPhysicalDeviceAccelerationStructurePropertiesKHR.prototype.maxInstanceCount BigInt Number is the maximum number of instances in the top level acceleration structure. VkPhysicalDeviceAccelerationStructurePropertiesKHR.prototype.maxPrimitiveCount BigInt Number is the maximum number of triangles or AABBs in all geometries in the bottom level acceleration structure. VkPhysicalDeviceAccelerationStructurePropertiesKHR.prototype.maxPerStageDescriptorAccelerationStructures Number is the maximum number of acceleration structure bindings that can be accessible to a single shader stage in a pipeline layout. Descriptor bindings with a descriptor type of VK_DESCRIPTOR_TYPE_ACCELERATION_STRUCTURE_KHR count against this limit. Only descriptor bindings in descriptor set layouts created without the VK_DESCRIPTOR_SET_LAYOUT_CREATE_UPDATE_AFTER_BIND_POOL_BIT bit set count against this limit. VkPhysicalDeviceAccelerationStructurePropertiesKHR.prototype.maxPerStageDescriptorUpdateAfterBindAccelerationStructures Number is similar to maxPerStageDescriptorAccelerationStructures but counts descriptor bindings from descriptor sets created with or without the VK_DESCRIPTOR_SET_LAYOUT_CREATE_UPDATE_AFTER_BIND_POOL_BIT bit set. VkPhysicalDeviceAccelerationStructurePropertiesKHR.prototype.maxDescriptorSetAccelerationStructures Number is the maximum number of acceleration structure descriptors that can be included in descriptor bindings in a pipeline layout across all pipeline shader stages and descriptor set numbers. Descriptor bindings with a descriptor type of VK_DESCRIPTOR_TYPE_ACCELERATION_STRUCTURE_KHR count against this limit. Only descriptor bindings in descriptor set layouts created without the VK_DESCRIPTOR_SET_LAYOUT_CREATE_UPDATE_AFTER_BIND_POOL_BIT bit set count against this limit. VkPhysicalDeviceAccelerationStructurePropertiesKHR.prototype.maxDescriptorSetUpdateAfterBindAccelerationStructures Number is similar to maxDescriptorSetAccelerationStructures but counts descriptor bindings from descriptor sets created with or without the VK_DESCRIPTOR_SET_LAYOUT_CREATE_UPDATE_AFTER_BIND_POOL_BIT bit set. VkPhysicalDeviceAccelerationStructurePropertiesKHR.prototype.minAccelerationStructureScratchOffsetAlignment Number is the minimum required alignment, in bytes, for scratch data passed in to an acceleration structure build command. Default Properties VkPhysicalDeviceAccelerationStructurePropertiesKHR.prototype.memoryBuffer ArrayBuffer Native memory reference of the structure. VkPhysicalDeviceAccelerationStructurePropertiesKHR.memoryLayout Object Object describing the structure's memory layout. VkPhysicalDeviceAccelerationStructurePropertiesKHR.byteLength Number Total native byte length of the structure. Internal Properties These properties are mainly used by nvk internally. VkPhysicalDeviceAccelerationStructurePropertiesKHR.reset Resets the native memory of the structure. VkPhysicalDeviceAccelerationStructurePropertiesKHR.flush Writes the structure's data into the native memory. VkPhysicalDeviceAccelerationStructurePropertiesKHR.reflect Synchronizes the structure's data with the native memory.
