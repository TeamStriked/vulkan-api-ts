Properties VkMemoryRequirements.prototype.size BigInt Number is the size, in bytes, of the memory allocation required for the resource. VkMemoryRequirements.prototype.alignment BigInt Number is the alignment, in bytes, of the offset within the allocation required for the resource. VkMemoryRequirements.prototype.memoryTypeBits Number is a bitmask and contains one bit set for every supported memory type for the resource. Bit `i` is set if and only if the memory type `i` in the VkPhysicalDeviceMemoryProperties structure for the physical device is supported for the resource. Default Properties VkMemoryRequirements.prototype.memoryBuffer ArrayBuffer Native memory reference of the structure. VkMemoryRequirements.memoryLayout Object Object describing the structure's memory layout. VkMemoryRequirements.byteLength Number Total native byte length of the structure. Internal Properties These properties are mainly used by nvk internally. VkMemoryRequirements.reset Resets the native memory of the structure. VkMemoryRequirements.flush Writes the structure's data into the native memory. VkMemoryRequirements.reflect Synchronizes the structure's data with the native memory.
