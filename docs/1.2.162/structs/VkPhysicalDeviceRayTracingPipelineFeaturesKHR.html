VkPhysicalDeviceRayTracingPipelineFeaturesKHR.prototype.rayTracingPipelineShaderGroupHandleCaptureReplay Boolean indicates whether the implementation supports saving and reusing shader group handles, e.g. for trace capture and replay. VkPhysicalDeviceRayTracingPipelineFeaturesKHR.prototype.rayTracingPipelineShaderGroupHandleCaptureReplayMixed Boolean indicates whether the implementation supports reuse of shader group handles being arbitrarily mixed with creation of non-reused shader group handles.
