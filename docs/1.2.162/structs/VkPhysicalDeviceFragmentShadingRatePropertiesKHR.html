VkPhysicalDeviceFragmentShadingRatePropertiesKHR.prototype.minFragmentShadingRateAttachmentTexelSize VkExtent2D indicates minimum supported width and height of the portion of the framebuffer corresponding to each texel in a fragment shading rate attachment. Each value must be less than or equal to the values in maxFragmentShadingRateAttachmentTexelSize. Each value must be a power-of-two. It must be [eq]#(0,0)# if the <attachmentFragmentShadingRate>> feature is not supported. VkPhysicalDeviceFragmentShadingRatePropertiesKHR.prototype.maxFragmentShadingRateAttachmentTexelSize VkExtent2D indicates maximum supported width and height of the portion of the framebuffer corresponding to each texel in a fragment shading rate attachment. Each value must be greater than or equal to the values in minFragmentShadingRateAttachmentTexelSize. Each value must be a power-of-two. It must be [eq]#(0,0)# if the <attachmentFragmentShadingRate>> feature is not supported. VkPhysicalDeviceFragmentShadingRatePropertiesKHR.prototype.maxFragmentShadingRateAttachmentTexelSizeAspectRatio Number indicates the maximum ratio between the width and height of the portion of the framebuffer corresponding to each texel in a fragment shading rate attachment. maxFragmentShadingRateAttachmentTexelSizeAspectRatio must be a power-of-two value, and must be less than or equal to [eq]#max(maxFragmentShadingRateAttachmentTexelSize.width / minFragmentShadingRateAttachmentTexelSize.height, maxFragmentShadingRateAttachmentTexelSize.height / minFragmentShadingRateAttachmentTexelSize.width)#. It must be 0 if the <attachmentFragmentShadingRate>> feature is not supported.
