Properties VkStencilOpState.prototype.failOp VkStencilOp is a VkStencilOp value specifying the action performed on samples that fail the stencil test. VkStencilOpState.prototype.passOp VkStencilOp is a VkStencilOp value specifying the action performed on samples that pass both the depth and stencil tests. VkStencilOpState.prototype.depthFailOp VkStencilOp is a VkStencilOp value specifying the action performed on samples that pass the stencil test and fail the depth test. VkStencilOpState.prototype.compareOp VkCompareOp is a VkCompareOp value specifying the comparison operator used in the stencil test. VkStencilOpState.prototype.compareMask Number selects the bits of the unsigned integer stencil values participating in the stencil test. VkStencilOpState.prototype.writeMask Number selects the bits of the unsigned integer stencil values updated by the stencil test in the stencil framebuffer attachment. VkStencilOpState.prototype.reference Number is a number reference value that is used in the unsigned stencil comparison.
