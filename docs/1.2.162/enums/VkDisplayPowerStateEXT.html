VkDisplayPowerStateEXT = {
  VK_DISPLAY_POWER_STATE_OFF_EXT: 0,
  VK_DISPLAY_POWER_STATE_SUSPEND_EXT: 1,
  VK_DISPLAY_POWER_STATE_ON_EXT: 2
};
 Properties VkDisplayPowerStateEXT.VK_DISPLAY_POWER_STATE_OFF_EXT specifies that the display is powered down. VkDisplayPowerStateEXT.VK_DISPLAY_POWER_STATE_SUSPEND_EXT specifies that the display is put into a low power mode, from which it may be able to transition back to VK_DISPLAY_POWER_STATE_ON_EXT more quickly than if it were in VK_DISPLAY_POWER_STATE_OFF_EXT. This state may be the same as VK_DISPLAY_POWER_STATE_OFF_EXT. VkDisplayPowerStateEXT.VK_DISPLAY_POWER_STATE_ON_EXT specifies that the display is powered on.
