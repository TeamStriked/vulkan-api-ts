Parameters object When instantiated with an object, members can be filled directly. Properties VkPhysicalDeviceMeshShaderPropertiesNV.prototype.sType VkStructureType is the type of this structure. VkPhysicalDeviceMeshShaderPropertiesNV.prototype.pNext null is null or a reference to an extension-specific structure. VkPhysicalDeviceMeshShaderPropertiesNV.prototype.maxDrawMeshTasksCount Number is the maximum number of local workgroups that can be launched by a single draw mesh tasks command. See <>. VkPhysicalDeviceMeshShaderPropertiesNV.prototype.maxTaskWorkGroupInvocations Number is the maximum total number of task shader invocations in a single local workgroup. The product of the X, Y, and Z sizes as specified by the LocalSize execution mode in shader modules and by the object decorated by the WorkgroupSize decoration must be less than or equal to this limit. VkPhysicalDeviceMeshShaderPropertiesNV.prototype.maxTaskWorkGroupSize Array[Number] VkPhysicalDeviceMeshShaderPropertiesNV.prototype.maxTaskTotalMemorySize Number is the maximum number of bytes that the task shader can use in total for shared and output memory combined. VkPhysicalDeviceMeshShaderPropertiesNV.prototype.maxTaskOutputCount Number is the maximum number of output tasks a single task shader workgroup can emit. VkPhysicalDeviceMeshShaderPropertiesNV.prototype.maxMeshWorkGroupInvocations Number is the maximum total number of mesh shader invocations in a single local workgroup. The product of the X, Y, and Z sizes as specified by the LocalSize execution mode in shader modules and by the object decorated by the WorkgroupSize decoration must be less than or equal to this limit. VkPhysicalDeviceMeshShaderPropertiesNV.prototype.maxMeshWorkGroupSize Array[Number] VkPhysicalDeviceMeshShaderPropertiesNV.prototype.maxMeshTotalMemorySize Number is the maximum number of bytes that the mesh shader can use in total for shared and output memory combined. VkPhysicalDeviceMeshShaderPropertiesNV.prototype.maxMeshOutputVertices Number is the maximum number of vertices a mesh shader output can store. VkPhysicalDeviceMeshShaderPropertiesNV.prototype.maxMeshOutputPrimitives Number is the maximum number of primitives a mesh shader output can store. VkPhysicalDeviceMeshShaderPropertiesNV.prototype.maxMeshMultiviewViewCount Number is the maximum number of multi-view views a mesh shader can use. VkPhysicalDeviceMeshShaderPropertiesNV.prototype.meshOutputPerVertexGranularity Number is the granularity with which mesh vertex outputs are allocated. The value can be used to compute the memory size used by the mesh shader, which must be less than or equal to maxMeshTotalMemorySize. VkPhysicalDeviceMeshShaderPropertiesNV.prototype.meshOutputPerPrimitiveGranularity Number is the granularity with which mesh outputs qualified as per-primitive are allocated. The value can be used to compute the memory size used by the mesh shader, which must be less than or equal to maxMeshTotalMemorySize.
