Parameters object When instantiated with an object, members can be filled directly. Properties VkSparseMemoryBind.prototype.resourceOffset BigInt Number is the offset into the resource. VkSparseMemoryBind.prototype.size BigInt Number is the size of the memory region to be bound. VkSparseMemoryBind.prototype.memory VkDeviceMemory is the VkDeviceMemory object that the range of the resource is bound to. If memory is VK_NULL_HANDLE, the range is unbound. VkSparseMemoryBind.prototype.memoryOffset BigInt Number is the offset into the VkDeviceMemory object to bind the resource range to. If memory is VK_NULL_HANDLE, this value is ignored. VkSparseMemoryBind.prototype.flags VkSparseMemoryBindFlagBits is a bitmask of VkSparseMemoryBindFlagBits specifying usage of the binding operation. Default Properties VkSparseMemoryBind.prototype.memoryBuffer ArrayBuffer Native memory reference of the structure. VkSparseMemoryBind.memoryLayout Object Object describing this structure's memory layout. VkSparseMemoryBind.byteLength Number Total native byte length of this structure.
