Properties VkPhysicalDeviceGroupProperties.prototype.sType VkStructureType is the type of this structure. VkPhysicalDeviceGroupProperties.prototype.pNext null is null or a reference to an extension-specific structure. VkPhysicalDeviceGroupProperties.prototype.physicalDeviceCount Number is the number of physical devices in the group. VkPhysicalDeviceGroupProperties.prototype.physicalDevices Array[VkPhysicalDevice] is an array of physical device handles representing all physical devices in the group. The first physicalDeviceCount elements of the array will be valid. VkPhysicalDeviceGroupProperties.prototype.subsetAllocation Boolean specifies whether logical devices created from the group support allocating device memory on a subset of devices, via the deviceMask member of the VkMemoryAllocateFlagsInfo. If this is VK_FALSE, then all device memory allocations are made across all physical devices in the group. If physicalDeviceCount is `1`, then subsetAllocation must be VK_FALSE. Default Properties VkPhysicalDeviceGroupProperties.prototype.memoryBuffer ArrayBuffer Native memory reference of the structure. VkPhysicalDeviceGroupProperties.memoryLayout Object Object describing this structure's memory layout. VkPhysicalDeviceGroupProperties.byteLength Number Total native byte length of this structure.
