Parameters object When instantiated with an object, members can be filled directly. Properties VkRayTracingShaderGroupCreateInfoNV.prototype.sType VkStructureType is the type of this structure. VkRayTracingShaderGroupCreateInfoNV.prototype.pNext null is null or a reference to an extension-specific structure. VkRayTracingShaderGroupCreateInfoNV.prototype.type VkRayTracingShaderGroupTypeNV is the type of hit group specified in this structure. VkRayTracingShaderGroupCreateInfoNV.prototype.generalShader Number is the index of the ray generation, miss, or callable shader from VkRayTracingPipelineCreateInfoNV::pStages in the group if the shader group has type of VK_RAY_TRACING_SHADER_GROUP_TYPE_GENERAL_NV and VK_SHADER_UNUSED_NV otherwise. VkRayTracingShaderGroupCreateInfoNV.prototype.closestHitShader Number is the optional index of the closest hit shader from VkRayTracingPipelineCreateInfoNV::pStages in the group if the shader group has type of VK_RAY_TRACING_SHADER_GROUP_TYPE_TRIANGLES_HIT_GROUP_NV or VK_RAY_TRACING_SHADER_GROUP_TYPE_PROCEDURAL_HIT_GROUP_NV and VK_SHADER_UNUSED_NV otherwise. VkRayTracingShaderGroupCreateInfoNV.prototype.anyHitShader Number is the optional index of the any-hit shader from VkRayTracingPipelineCreateInfoNV::pStages in the group if the shader group has type of VK_RAY_TRACING_SHADER_GROUP_TYPE_TRIANGLES_HIT_GROUP_NV or VK_RAY_TRACING_SHADER_GROUP_TYPE_PROCEDURAL_HIT_GROUP_NV and VK_SHADER_UNUSED_NV otherwise. VkRayTracingShaderGroupCreateInfoNV.prototype.intersectionShader Number is the index of the intersection shader from VkRayTracingPipelineCreateInfoNV::pStages in the group if the shader group has type of VK_RAY_TRACING_SHADER_GROUP_TYPE_PROCEDURAL_HIT_GROUP_NV and VK_SHADER_UNUSED_NV otherwise.
