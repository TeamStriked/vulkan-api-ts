 Categories vkGetDisplayPlaneCapabilitiesKHR Query capabilities of a mode and plane combination Syntax 
VkResult vkGetDisplayPlaneCapabilitiesKHR();
Parameters physicalDevice VkPhysicalDevice is the physical device associated with display mode VkDisplayModeKHR is the display mode the application intends to program when using the specified plane. Note this parameter also implicitly specifies a display. planeIndex Number is the plane which the application intends to use with the display, and is less than the number of display planes supported by the device. pCapabilities VkDisplayPlaneCapabilitiesKHR is a reference to a VkDisplayPlaneCapabilitiesKHR structure in which the capabilities are returned.
