 Categories vkCmdDrawIndexedIndirectCountAMD Perform an indexed indirect draw with the draw count sourced from a buffer Syntax 
void vkCmdDrawIndexedIndirectCountAMD();
Parameters commandBuffer VkCommandBuffer is the command buffer into which the command is recorded. buffer VkBuffer where parameters begin. offset BigInt Number countBuffer VkBuffer where the draw count begins. countBufferOffset BigInt Number maxDrawCount Number specifies the maximum number of draws that will be executed. The actual number of executed draw calls is the minimum of the count specified in countBuffer and maxDrawCount. stride Number is the byte stride between successive sets of draw parameters.
