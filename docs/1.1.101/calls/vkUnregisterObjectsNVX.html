 Categories vkUnregisterObjectsNVX Unregister resource bindings in an object table Syntax 
VkResult vkUnregisterObjectsNVX();
Parameters device VkDevice is the logical device that creates the object table. objectTable VkObjectTableNVX is the table from which the resources are unregistered. objectCount Number is the number of resources being removed from the object table. pObjectEntryTypes Int32Array pObjectIndices Uint32Array provides the array of object indices to be removed.
