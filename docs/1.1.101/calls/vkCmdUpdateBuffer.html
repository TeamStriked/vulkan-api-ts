 Categories vkCmdUpdateBuffer Update a buffer\ Syntax 
void vkCmdUpdateBuffer();
Parameters commandBuffer VkCommandBuffer is the command buffer into which the command will be recorded. dstBuffer VkBuffer is a handle to the buffer to be updated. dstOffset BigInt Number is the byte offset into the buffer to start updating, and must be a multiple of 4. dataSize BigInt Number is the number of bytes to update, and must be a multiple of 4. pData ArrayBuffer is a reference to the source data for the buffer update, and must be at least dataSize bytes in size.
