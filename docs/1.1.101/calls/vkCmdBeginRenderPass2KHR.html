Parameters commandBuffer VkCommandBuffer is the command buffer in which to record the command. pRenderPassBegin VkRenderPassBeginInfo is a reference to a VkRenderPassBeginInfo structure (defined below) which indicates the render pass to begin an instance of, and the framebuffer the instance uses. pSubpassBeginInfo VkSubpassBeginInfoKHR is a reference to a VkSubpassBeginInfoKHR structure which contains information about the subpass which is about to begin rendering.
