 Categories vkCmdBuildAccelerationStructureNV Build an acceleration structure Syntax 
void vkCmdBuildAccelerationStructureNV();
Parameters commandBuffer VkCommandBuffer is the command buffer into which the command will be recorded. pInfo VkAccelerationStructureInfoNV contains the shared information for the acceleration structure's structure. instanceData VkBuffer is the buffer containing instance data that will be used to build the acceleration structure as described in <> This parameter must be null for bottom level acceleration structures. instanceOffset BigInt Number is the offset in bytes (relative to the start of instanceData) at which the instance data is located. update Boolean dst VkAccelerationStructureNV reference to the target acceleration structure for the build. src VkAccelerationStructureNV reference to an existing acceleration structure that is to be used to update the dst acceleration structure. scratch VkBuffer is the VkBuffer that will be used as scratch memory for the build. scratchOffset BigInt Number is the offset in bytes relative to the start of scratch that will be used as a scratch memory.
