 Properties VkPolygonMode.VK_POLYGON_MODE_FILL specifies that polygons are rendered using the polygon rasterization rules in this section. VkPolygonMode.VK_POLYGON_MODE_LINE specifies that polygon edges are drawn as line segments. VkPolygonMode.VK_POLYGON_MODE_POINT specifies that polygon vertices are drawn as reference.
