Properties VkPhysicalDeviceShaderCorePropertiesAMD.prototype.sType VkStructureType is the type of this structure. VkPhysicalDeviceShaderCorePropertiesAMD.prototype.pNext null is null or a reference to an extension-specific structure. VkPhysicalDeviceShaderCorePropertiesAMD.prototype.shaderEngineCount Number is a number value indicating the number of shader engines found inside the shader core of the physical device. VkPhysicalDeviceShaderCorePropertiesAMD.prototype.shaderArraysPerEngineCount Number is a number value indicating the number of shader arrays inside a shader engine. Each shader array has its own scan converter, set of compute units, and a render back end (color and depth buffers). Shader arrays within a shader engine share shader processor input (wave launcher) and shader export (export buffer) units. Currently, a shader engine can have one or two shader arrays. VkPhysicalDeviceShaderCorePropertiesAMD.prototype.computeUnitsPerShaderArray Number is a number value indicating the number of compute units within a shader array. A compute unit houses a set of SIMDs along with a sequencer module and a local data store. VkPhysicalDeviceShaderCorePropertiesAMD.prototype.simdPerComputeUnit Number is a number value indicating the number of SIMDs inside a compute unit. Each SIMD processes a single instruction at a time. VkPhysicalDeviceShaderCorePropertiesAMD.prototype.wavefrontsPerSimd Number VkPhysicalDeviceShaderCorePropertiesAMD.prototype.wavefrontSize Number is a number value indicating the number of channels (or threads) in a wavefront. VkPhysicalDeviceShaderCorePropertiesAMD.prototype.sgprsPerSimd Number is a number value indicating the number of physical Scalar General Purpose Registers (SGPRs) per SIMD. VkPhysicalDeviceShaderCorePropertiesAMD.prototype.minSgprAllocation Number is a number value indicating the minimum number of SGPRs allocated for a wave. VkPhysicalDeviceShaderCorePropertiesAMD.prototype.maxSgprAllocation Number is a number value indicating the maximum number of SGPRs allocated for a wave. VkPhysicalDeviceShaderCorePropertiesAMD.prototype.sgprAllocationGranularity Number is a number value indicating the granularity of SGPR allocation for a wave. VkPhysicalDeviceShaderCorePropertiesAMD.prototype.vgprsPerSimd Number is a number value indicating the number of physical Vector General Purpose Registers (VGPRs) per SIMD. VkPhysicalDeviceShaderCorePropertiesAMD.prototype.minVgprAllocation Number is a number value indicating the minimum number of VGPRs allocated for a wave. VkPhysicalDeviceShaderCorePropertiesAMD.prototype.maxVgprAllocation Number is a number value indicating the maximum number of VGPRs allocated for a wave. VkPhysicalDeviceShaderCorePropertiesAMD.prototype.vgprAllocationGranularity Number is a number value indicating the granularity of VGPR allocation for a wave.
