Parameters object When instantiated with an object, members can be filled directly. Properties VkClearAttachment.prototype.aspectMask VkImageAspectFlagBits is a mask selecting the color, depth and/or stencil aspects of the attachment to be cleared. VkClearAttachment.prototype.colorAttachment Number is only meaningful if VK_IMAGE_ASPECT_COLOR_BIT is set in aspectMask, in which case it is an index to the pColorAttachments array in the VkSubpassDescription structure of the current subpass which selects the color attachment to clear. VkClearAttachment.prototype.clearValue VkClearValue is the color or depth/stencil value to clear the attachment to, as described in <> below.
