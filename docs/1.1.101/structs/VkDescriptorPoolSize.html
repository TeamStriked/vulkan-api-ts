Parameters object When instantiated with an object, members can be filled directly. Properties VkDescriptorPoolSize.prototype.type VkDescriptorType is the type of descriptor. VkDescriptorPoolSize.prototype.descriptorCount Number is the number of descriptors of that type to allocate. If type is VK_DESCRIPTOR_TYPE_INLINE_UNIFORM_BLOCK_EXT then descriptorCount is the number of bytes to allocate for descriptors of this type.
