Parameters object When instantiated with an object, members can be filled directly. Properties VkFramebufferCreateInfo.prototype.sType VkStructureType is the type of this structure. VkFramebufferCreateInfo.prototype.pNext null is null or a reference to an extension-specific structure. VkFramebufferCreateInfo.prototype.flags Number is reserved for future use. VkFramebufferCreateInfo.prototype.renderPass VkRenderPass is a render pass that defines what render passes the framebuffer will be compatible with. See <> for details. VkFramebufferCreateInfo.prototype.attachmentCount Number is the number of attachments. VkFramebufferCreateInfo.prototype.pAttachments Array[VkImageView] is an array of VkImageView handles, each of which will be used as the corresponding attachment in a render pass instance. VkFramebufferCreateInfo.prototype.width Number VkFramebufferCreateInfo.prototype.height Number VkFramebufferCreateInfo.prototype.layers Number define the dimensions of the framebuffer. If the render pass uses multiview, then layers must be one and each attachment requires a number of layers that is greater than the maximum bit index set in the view mask in the subpasses in which it is used. Default Properties VkFramebufferCreateInfo.prototype.memoryBuffer ArrayBuffer Native memory reference of the structure. VkFramebufferCreateInfo.memoryLayout Object Object describing this structure's memory layout. VkFramebufferCreateInfo.byteLength Number Total native byte length of this structure.
