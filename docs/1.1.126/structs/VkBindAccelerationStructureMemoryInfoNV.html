Parameters object When instantiated with an object, members can be filled directly. Properties VkBindAccelerationStructureMemoryInfoNV.prototype.sType VkStructureType is the type of this structure. VkBindAccelerationStructureMemoryInfoNV.prototype.pNext null is null or a reference to an extension-specific structure. VkBindAccelerationStructureMemoryInfoNV.prototype.accelerationStructure VkAccelerationStructureNV is the acceleration structure to be attached to memory. VkBindAccelerationStructureMemoryInfoNV.prototype.memory VkDeviceMemory is a VkDeviceMemory object describing the device memory to attach. VkBindAccelerationStructureMemoryInfoNV.prototype.memoryOffset BigInt Number is the start offset of the region of memory that is to be bound to the acceleration structure. The number of bytes returned in the VkMemoryRequirements::size member in memory, starting from memoryOffset bytes, will be bound to the specified acceleration structure. VkBindAccelerationStructureMemoryInfoNV.prototype.deviceIndexCount Number is the number of elements in pDeviceIndices. VkBindAccelerationStructureMemoryInfoNV.prototype.pDeviceIndices Uint32Array is an array of device indices.
