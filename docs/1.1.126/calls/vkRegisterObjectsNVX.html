 Categories vkRegisterObjectsNVX Register resource bindings in an object table Syntax 
VkResult vkRegisterObjectsNVX();
Parameters device VkDevice is the logical device that creates the object table. objectTable VkObjectTableNVX is the table for which the resources are registered. objectCount Number is the number of resources to register. ppObjectTableEntries Array[VkObjectTableEntryNVX] provides an array for detailed binding informations. Each array element is a reference to a structure of type VkObjectTablePipelineEntryNVX, VkObjectTableDescriptorSetEntryNVX, VkObjectTableVertexBufferEntryNVX, VkObjectTableIndexBufferEntryNVX or VkObjectTablePushConstantEntryNVX (see below for details). pObjectIndices Uint32Array are the indices at which each resource is registered.
