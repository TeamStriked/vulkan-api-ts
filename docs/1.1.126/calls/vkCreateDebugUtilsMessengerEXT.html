Parameters instance VkInstance the instance the messenger will be used with. pCreateInfo VkDebugUtilsMessengerCreateInfoEXT is a reference to a VkDebugUtilsMessengerCreateInfoEXT structure containing the callback reference, as well as defining conditions under which this messenger will trigger the callback. pAllocator VkAllocationCallbacks controls host memory allocation as described in the <> chapter. pMessenger VkDebugUtilsMessengerEXT is a reference to a VkDebugUtilsMessengerEXT handle in which the created object is returned.
