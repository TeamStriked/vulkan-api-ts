 Categories vkGetPhysicalDeviceMultisamplePropertiesEXT Report sample count specific multisampling capabilities of a physical device Syntax 
void vkGetPhysicalDeviceMultisamplePropertiesEXT();
Parameters physicalDevice VkPhysicalDevice is the physical device from which to query the additional multisampling capabilities. samples VkSampleCountFlagBits is the sample count to query the capabilities for. pMultisampleProperties VkMultisamplePropertiesEXT is a reference to a VkMultisamplePropertiesEXT structure in which information about the additional multisampling capabilities specific to the sample count is returned.
