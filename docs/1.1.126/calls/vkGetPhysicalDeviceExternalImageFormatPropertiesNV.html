 Categories vkGetPhysicalDeviceExternalImageFormatPropertiesNV determine image capabilities compatible with external memory handle types Syntax 
VkResult vkGetPhysicalDeviceExternalImageFormatPropertiesNV();
Parameters physicalDevice VkPhysicalDevice is the physical device from which to query the image capabilities format VkFormat is the image format, corresponding to VkImageCreateInfo::format. type VkImageType is the image type, corresponding to VkImageCreateInfo::imageType. tiling VkImageTiling is the image tiling, corresponding to VkImageCreateInfo::tiling. usage VkImageUsageFlagBits is the intended usage of the image, corresponding to VkImageCreateInfo::usage. flags VkImageCreateFlagBits is a bitmask describing additional parameters of the image, corresponding to VkImageCreateInfo::flags. externalHandleType VkExternalMemoryHandleTypeFlagBitsNV is either one of the bits from VkExternalMemoryHandleTypeFlagBitsNV, or 0. pExternalImageFormatProperties VkExternalImageFormatPropertiesNV is a reference to a VkExternalImageFormatPropertiesNV structure in which capabilities are returned.
