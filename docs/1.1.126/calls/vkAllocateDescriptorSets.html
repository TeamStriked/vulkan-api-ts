 Categories vkAllocateDescriptorSets Allocate one or more descriptor sets Syntax 
VkResult vkAllocateDescriptorSets();
Parameters device VkDevice is the logical device that owns the descriptor pool. pAllocateInfo VkDescriptorSetAllocateInfo is a reference to a VkDescriptorSetAllocateInfo structure describing parameters of the allocation. pDescriptorSets Array[VkDescriptorSet] is an array of VkDescriptorSet handles in which the resulting descriptor set objects are returned.
