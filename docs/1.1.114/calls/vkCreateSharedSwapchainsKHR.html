VkResult vkCreateSharedSwapchainsKHR();
Parameters device VkDevice is the device to create the swapchains for. swapchainCount Number is the number of swapchains to create. pCreateInfos Array[VkSwapchainCreateInfoKHR] is an array of VkSwapchainCreateInfoKHR structures specifying the parameters of the created swapchains. pAllocator VkAllocationCallbacks is the allocator used for host memory allocated for the swapchain objects when there is no more specific allocator available (see <>). pSwapchains Array[VkSwapchainKHR] is an array of VkSwapchainKHR handles in which the created swapchain objects will be returned.
