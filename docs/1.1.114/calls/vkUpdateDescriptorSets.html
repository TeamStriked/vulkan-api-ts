 Categories vkUpdateDescriptorSets Update the contents of a descriptor set object Syntax 
void vkUpdateDescriptorSets();
Parameters device VkDevice is the logical device that updates the descriptor sets. descriptorWriteCount Number is the number of elements in the pDescriptorWrites array. pDescriptorWrites Array[VkWriteDescriptorSet] is an array of VkWriteDescriptorSet structures describing the descriptor sets to write to. descriptorCopyCount Number is the number of elements in the pDescriptorCopies array. pDescriptorCopies Array[VkCopyDescriptorSet] is an array of VkCopyDescriptorSet structures describing the descriptor sets to copy between.
