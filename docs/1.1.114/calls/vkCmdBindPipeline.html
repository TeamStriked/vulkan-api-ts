 Categories vkCmdBindPipeline Bind a pipeline object to a command buffer Syntax 
void vkCmdBindPipeline();
Parameters commandBuffer VkCommandBuffer is the command buffer that the pipeline will be bound to. pipelineBindPoint VkPipelineBindPoint is a VkPipelineBindPoint value specifying whether to bind to the compute or graphics bind point. Binding one does not disturb the other. pipeline VkPipeline is the pipeline to be bound.
