VkPhysicalDeviceFragmentDensityMapFeaturesEXT.prototype.fragmentDensityMapDynamic Boolean specifies whether the implementation supports dynamic fragment density map image views. If this feature is not enabled, VK_IMAGE_VIEW_CREATE_FRAGMENT_DENSITY_MAP_DYNAMIC_BIT_EXT must not be included in VkImageViewCreateInfo::flags. VkPhysicalDeviceFragmentDensityMapFeaturesEXT.prototype.fragmentDensityMapNonSubsampledImages Boolean specifies whether the implementation supports regular non-subsampled image attachments with fragment density map render passes.
