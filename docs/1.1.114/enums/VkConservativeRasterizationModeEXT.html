 Categories VkConservativeRasterizationModeEXT Specify the conservative rasterization mode Stub 
VkConservativeRasterizationModeEXT = {
  VK_CONSERVATIVE_RASTERIZATION_MODE_DISABLED_EXT: 0,
  VK_CONSERVATIVE_RASTERIZATION_MODE_OVERESTIMATE_EXT: 1,
  VK_CONSERVATIVE_RASTERIZATION_MODE_UNDERESTIMATE_EXT: 2
};
 Properties VkConservativeRasterizationModeEXT.VK_CONSERVATIVE_RASTERIZATION_MODE_DISABLED_EXT specifies that conservative rasterization is disabled and rasterization proceeds as normal. VkConservativeRasterizationModeEXT.VK_CONSERVATIVE_RASTERIZATION_MODE_OVERESTIMATE_EXT specifies that conservative rasterization is enabled in overestimation mode. VkConservativeRasterizationModeEXT.VK_CONSERVATIVE_RASTERIZATION_MODE_UNDERESTIMATE_EXT specifies that conservative rasterization is enabled in underestimation mode.
