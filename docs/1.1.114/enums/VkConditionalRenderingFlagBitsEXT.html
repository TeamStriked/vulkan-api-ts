 Categories VkConditionalRenderingFlagBitsEXT Specify the behavior of conditional rendering Stub 
VkConditionalRenderingFlagBitsEXT = {
  VK_CONDITIONAL_RENDERING_INVERTED_BIT_EXT: 0x1
};
 Properties VkConditionalRenderingFlagBitsEXT.VK_CONDITIONAL_RENDERING_INVERTED_BIT_EXT specifies the condition used to determine whether to discard rendering commands or not. That is, if the 32-bit predicate read from buffer memory at offset is zero, the rendering commands are not discarded, and if non zero, then they are discarded.
