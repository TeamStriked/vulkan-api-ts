VkAttachmentSampleLocationsEXT.prototype.sampleLocationsInfo VkSampleLocationsInfoEXT is the sample locations state to use for the layout transition of the given attachment from the initial layout of the attachment to the image layout specified for the attachment in the first subpass using it.
