Properties VkLayerProperties.prototype.layerName String is a string specifying the name of the layer. Use this name in the ppEnabledLayerNames array passed in the VkInstanceCreateInfo structure to enable this layer for an instance. VkLayerProperties.prototype.specVersion Number is the Vulkan version the layer was written to, encoded as described in <>. VkLayerProperties.prototype.implementationVersion Number is the version of this layer. It is a number, increasing with backward compatible changes. VkLayerProperties.prototype.description String is a string providing additional details that can be used by the application to identify the layer. Default Properties VkLayerProperties.prototype.memoryBuffer ArrayBuffer Native memory reference of the structure. VkLayerProperties.memoryLayout Object Object describing the structure's memory layout. VkLayerProperties.byteLength Number Total native byte length of the structure. Internal Properties These properties are mainly used by nvk internally. VkLayerProperties.reset Resets the native memory of the structure. VkLayerProperties.flush Writes the structure's data into the native memory. VkLayerProperties.reflect Synchronizes the structure's data with the native memory.
