Parameters object When instantiated with an object, members can be filled directly. Properties VkImageViewCreateInfo.prototype.sType VkStructureType is the type of this structure. VkImageViewCreateInfo.prototype.pNext null is null or a reference to an extension-specific structure. VkImageViewCreateInfo.prototype.flags VkImageViewCreateFlagBits is a bitmask of VkImageViewCreateFlagBits describing additional parameters of the image view. VkImageViewCreateInfo.prototype.image VkImage is a VkImage on which the view will be created. VkImageViewCreateInfo.prototype.viewType VkImageViewType is a VkImageViewType value specifying the type of the image view. VkImageViewCreateInfo.prototype.format VkFormat is a VkFormat describing the format and type used to interpret texel blocks in the image. VkImageViewCreateInfo.prototype.components VkComponentMapping is a VkComponentMapping specifies a remapping of color components (or of depth or stencil components after they have been converted into color components). VkImageViewCreateInfo.prototype.subresourceRange VkImageSubresourceRange is a VkImageSubresourceRange selecting the set of mipmap levels and array layers to be accessible to the view. Default Properties VkImageViewCreateInfo.prototype.memoryBuffer ArrayBuffer Native memory reference of the structure. VkImageViewCreateInfo.memoryLayout Object Object describing the structure's memory layout. VkImageViewCreateInfo.byteLength Number Total native byte length of the structure. Internal Properties These properties are mainly used by nvk internally. VkImageViewCreateInfo.reset Resets the native memory of the structure.
