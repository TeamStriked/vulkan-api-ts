Parameters object When instantiated with an object, members can be filled directly. Properties VkPipelineColorBlendAttachmentState.prototype.blendEnable Boolean controls whether blending is enabled for the corresponding color attachment. If blending is not enabled, the source fragment's color for that attachment is passed through unmodified. VkPipelineColorBlendAttachmentState.prototype.srcColorBlendFactor VkBlendFactor selects which blend factor is used to determine the source factors [eq]#(S~r~,S~g~,S~b~)#. VkPipelineColorBlendAttachmentState.prototype.dstColorBlendFactor VkBlendFactor selects which blend factor is used to determine the destination factors [eq]#(D~r~,D~g~,D~b~)#. VkPipelineColorBlendAttachmentState.prototype.colorBlendOp VkBlendOp selects which blend operation is used to calculate the RGB values to write to the color attachment. VkPipelineColorBlendAttachmentState.prototype.srcAlphaBlendFactor VkBlendFactor selects which blend factor is used to determine the source factor [eq]#S~a~#. VkPipelineColorBlendAttachmentState.prototype.dstAlphaBlendFactor VkBlendFactor selects which blend factor is used to determine the destination factor [eq]#D~a~#. VkPipelineColorBlendAttachmentState.prototype.alphaBlendOp VkBlendOp selects which blend operation is use to calculate the alpha values to write to the color attachment. VkPipelineColorBlendAttachmentState.prototype.colorWriteMask VkColorComponentFlagBits is a bitmask of VkColorComponentFlagBits specifying which of the R, G, B, and/or A components are enabled for writing, as described for the <>.
