 Categories vkGetQueryPoolResults Copy results of queries in a query pool to a host memory region Syntax 
VkResult vkGetQueryPoolResults();
Parameters device VkDevice is the logical device that owns the query pool. queryPool VkQueryPool is the query pool managing the queries containing the desired results. firstQuery Number is the initial query index. queryCount Number is the number of queries to read. dataSize BigInt Number is the size in bytes of the buffer pointed to by pData. pData ArrayBuffer is a reference to a user-allocated buffer where the results will be written stride BigInt Number is the stride in bytes between results for individual queries within pData. flags VkQueryResultFlagBits is a bitmask of VkQueryResultFlagBits specifying how and when results are returned.
