 Categories vkCmdFillBuffer Fill a region of a buffer with a fixed value Syntax 
void vkCmdFillBuffer();
Parameters commandBuffer VkCommandBuffer is the command buffer into which the command will be recorded. dstBuffer VkBuffer is the buffer to be filled. dstOffset BigInt Number is the byte offset into the buffer at which to start filling, and must be a multiple of 4. size BigInt Number is the number of bytes to fill, and must be either a multiple of 4, or VK_WHOLE_SIZE to fill the range from offset to the end of the buffer. If VK_WHOLE_SIZE is used and the remaining size of the buffer is not a multiple of 4, then the nearest smaller multiple is used. data Number is the 4-byte word written repeatedly to the buffer to fill size bytes of data. The data word is written to memory according to the host endianness.
