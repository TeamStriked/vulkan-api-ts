 Categories VkSubpassContents Specify how commands in the first subpass of a render pass are provided Stub 
VkSubpassContents = {
  VK_SUBPASS_CONTENTS_INLINE: 0,
  VK_SUBPASS_CONTENTS_SECONDARY_COMMAND_BUFFERS: 1
};
 Properties VkSubpassContents.VK_SUBPASS_CONTENTS_INLINE specifies that the contents of the subpass will be recorded inline in the primary command buffer, and secondary command buffers must not be executed within the subpass. VkSubpassContents.VK_SUBPASS_CONTENTS_SECONDARY_COMMAND_BUFFERS specifies that the contents are recorded in secondary command buffers that will be called from the primary command buffer, and vkCmdExecuteCommands is the only valid command on the command buffer until vkCmdNextSubpass or vkCmdEndRenderPass.
